# Panduan Membuat Streamlit ✨

## Dataset
```
1. Data Penduduk
2. Kamus(dipisah-pisah berdasarkan daerah)
3. Koordinat
4. Kamus bahsasa daerah
5. Bonus ada tambahan gamabr ketiga daerah
```

## Alur
```
1. Munculkan peta dari ketiga daerah menggunakan GeoPandas atau GeoJson
2. Masukkan data koordinat
3. Dibuat sidebar dengan menampilkan markdown daerah yang dipilih
  3.1. Aceh dengan jumlah penduduk, koordinatnya, kamus, dan berada di pulau mana(Munculkan)
  3.2. Yogyakarta dengan jumlah penduduk, koordinat, kamus, dan berada di pulau mana(Munculkan)
  3.3. Nusa Tenggara Timur dengan jumlah penduduk, koordinat, kamus, dan berada di pulau mana
```
## Run steamlit app
```
streamlit run dashboard.py
```
